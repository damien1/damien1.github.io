Cyanogenmod 7.1 RC1

Monday, Jun 27, 2011| Tags: Android, Cyanogen

Have you updated? – the first release candidate for Cyanogenmod v7.1 has been pushed out today. This update adds (more) support for Android 2.3.4 a number of Android devices from Samsung, LG, Motorola and the HTC Incredible.

Here’s a summary of nice features from the changelog:

Common: Support for revoking application permissions

Common: Latest Superuser app

Common: Control brightness by sliding on statusbar

Common: ADWLauncher- Transitions, catalog improvements, bugfixes

Common: Optional GMT correction for SMS/MMS timestamps

 

You can download it via the Clockwork Rom Manager or manually.

Being a release candidate this should be fairly stable. If you’re new to rooting your phone or concerned about updating to this RC then wait for the next proper release.

 

Keep a backup of your SD card

Always, always, always before messing with new versions of rom – backup your SD card to a safe place.

Even if you’re good at roms and mods – backup all folders like Clockwork and the files like update.zip to a safe place
