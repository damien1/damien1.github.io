WordPress just updated to 3.2.1

Sunday, Jul 17, 2011| Tags: Releases

WordPress just released 3.2.1 so that means it’s time to update your WordPress install. This is a minor update and only a week since 3.2 was launched..

What’s in this release?

There isn’t much from a user or admin perspective, but you don’t normally expect that in a point release.

The main deliverables in the release are:

•Changes to the default Twenty Eleven theme

•some RTL (right to left) fixes.

• set browser default font to sans-serif.

How to Update WordPress safe & easy

This is very easy to do from any wp-admin page –  but you should always follow a simple process

  1. Make sure you have a current backup of your WordPress database
  2. Enable your Maintenance mode page.
  3. Disable any / all plugins
  4. Run update
  5. Update any plugins that also got updated
  6. Enable all plugins that you need
  7. Check for any new issues
  8. Disable your maintenance mode
