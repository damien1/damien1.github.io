Isotope Visual Layouts plugin for WordPress v1 is finally being tested

Monday, Nov 26, 2012| Tags: Isotope

The next version of my Isotope Visual Layouts Plugin for WordPress is finally in testing.  That’s to say, today I have finished all the development work I want to do for version 1.0.

There have been more than 500 downloads of the current version (which is great!) and so I’m excited to be sharing the news about the next release.

 

What’s in Version 1.0?

The 2 key features in this next version are:

  • 7 Different colour styles to choose from
  • Choose to include the feature image, image only or text only

The plugin has also been re-written to work on any page, post or custom-post - and it has been tested on WordPress Multisite.

 

When’s it due?

All going well … I just have to update a few test scripts this week and run them. So maybe next weekend.
