iWork &#8211; Pages I don&#039;t like

Tuesday, Oct 5, 2010| Tags: Apple, apps, iWork

I started the week with the ambition to just use iWork for a week. Today’s Tuesday and now I see the #failure of such a challenge.

iWork Pages can’t convert Word documents so easily

I’ve been sent a word document to review for a friend – that should be easy as iWork opens Microsoft Office Word .docx formats.  The document opens ok … but the content has been placed into separate textblocks on the page just like any traditional desktop publishing app would do – but of course this was supposed to be a document that needed to be cleaned up.

A simple step (select all and change the font face) was made impossible due as Select All wont select all text blocks. So it will be quicker for me to go install Office for Mac and do this in the document.

iWork Keynote now this works to convert PowerPoint

Forced to use Keynote – then I was surprised and got more than I expected from using Keynote with an existing PowerPoint presentation. I think I’ll continue to give this a try!
