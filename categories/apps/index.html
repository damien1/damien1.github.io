App Design Review of Evernote's First Time User Guide

Monday, May 21, 2012 by Damien Saunders design, Evernote, user experience

It’s not often that we (as app user or developers) stop to consider how important the First Time User experience or the Getting Started Guide is. If you had never heard of Evernote (or many other apps) then you rely on this introduction to do a lot of things. To sum it up, the first time user experience has to do two things: sell the customer that the app will do what the user thinks it will do (they’ve already installed it but app store descriptions can be misleading), and; tell the customer how to use it Ultimately, large multi-platform applications have many, many, more features then there is time for a Quick Start Guide or First Time User experience.

I know I’m going to lose a few ‘friends’ here .. but bare with me a minute.
