Isotope Visual Layouts Pro plugin for WordPress switches to Twitter Bootstrap

Tuesday, Mar 19, 2013| Tags: Isotope, Twitter Bootstrap

Isotope Visual Layouts Pro WordPress plugin is easier to use now that it’s using the Twitter Bootstrap framework with great CSS options and features.

Damien’s WordPress plugins have great features that mean you can improve your website and not change your theme. This is ideal for a designer or developer who wants to change the look n feel via custom CSS. Overtime, I’ve come to realise that most people download or buy the plugin on the basis off the look. So I’ve decided to switch to Twitter Bootstrap. This has a look n feel that you’ll have seen on many websites and additionally, it means I can focus on function.

If you’re into bootstrapping a website and wanting some great CSS styles and JQuery plugins, then you’ll already know about frameworks like Twitter Bootstrap.
