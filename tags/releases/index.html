WordPress Security make sure you update today

Monday, Apr 23, 2012 by Damien Saunders hack, malware, Releases, Security

When ever there is a security updatefor WordPress you should always update your blog and website as soon as possible. According to Kasperskyone of the reasons that the recent Flashback Mac OSx Trojan malware infected 600,000+ computers was due to infected WordPress websites. So how did this happen? When developers build you a website there typically is no time or budget left over for maintenance and updates. This means websites can be left running on old versions of WordPress with known security issues and these can be exploited by hackers.
