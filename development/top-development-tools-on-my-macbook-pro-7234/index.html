Top Development Tools on my MacBook Pro

Monday, Jun 11, 2012| Tags: Apple, apps

If you wanted to know what I apps and software I use every day then here you go … For starters this is all Mac OSx stuff.

  • AxureRP 6.5 – wireframes and interactive prototypes. UX/UI Heaven
  • Terminal – go command line (for SSH and most server stuff like testing with siege)
  • Coda 2 – An all in one IDE – FTP, MySQL database, CSS, Editor and more …
  • Transmit – only if I can’t be bothered to use Coda
  • Firefox (and then only for these 2 extensions)
  • SQLite Manager – SQLite may be the new mySQL … we’ll see
  • Selenium IDE – for browser based testing (but only in Firefox)
  • Sequel Pro – mySQL database manager
  • SourceTree (as it works with Bitbucket)

Other apps to know about

  • MarsEdit – Best blog tool around for WordPress – but definitely room to improve
  • Scrivener – for writing all that blog stuff
  • Omnigraffle – when you can’t use Axure (and don’t have Visio)
  • Pixelmator – who needs Photoshop
  • Aperture – who needs Lightroom
  • Netnewswire – I do all my RSS reading on the mobile now

Cloud based stuff I use every day

  • Google.apps – mail, domain stuff, analytics, Drive and Plus
  • Zoho Books – accounting for my business
  • JIRA / Atlassian
  • Trello – project management and scrum /agile boards
  • Last.fm – music history at its best
  • Bit.ly – URL and Analytics
  • Delicious – only when I don’t have a Chrome browser open
  • Bitbucket (cause it works with JIRA too)
